Get Started
1. Track code and data
$ git add train.py
$ dvc add images.zip
1
2. Connect code and data by commands
$ dvc run -d images.zip -o images/ unzip -q images.zip
$ dvc run -d images/ -d train.py -o model.p python train.py
2
Make changes and reproduce
$ vi train.py
$ dvc repro
