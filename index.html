Get Started
Track large data file, outside of in Git repository
$ dvc add images.zip
1
Define ML steps
$ dvc run -d images.zip -o images/ unzip -q images.zip
$ dvc run -d images/ -d train.py -o model.p python train.py
2
Make changes and regenerate affected steps
$ vi train.py
$ dvc repro
3
Share code and DVC metadata
$ git add .
$ git commit -m 'The baseline model'
$ git push
4
Share data and models
$ dvc config AWS.StoragePath mybucket/image_cnn
$ dvc push
